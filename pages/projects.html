Low-latency trading in C++

I took on this project to dive deeper into C++20 features and learn low-latency programming. Following a book, I documented each class and code component to reinforce my understanding. I explored the structure of a basic electronic trading ecosystem and implemented strategies like market-making and liquidity removal. The project covered multi-threading, memory pools, lock-free queues, low-latency logging, and TCP socket programming. I also containerized the application in Docker, almost just for fun, and to learn how to use it along the way.

A digital audio synthesizer in C++ (Klang)

In this project I ventured myself into the fields of DSP and audio programming, by building a plugin in Klang Studio (C++) based on FM operator synthesis. The goal was to explore how multiple operators can be combined and modulated to generate complex sounds, while working within the constraints of real-time audio systems. Along the way, I focused on practical aspects of audio development such as low-latency processing and safe memory management, making use of C++ smart pointers to handle resources... smartly ;).
